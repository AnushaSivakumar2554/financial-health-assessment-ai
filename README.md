# Financial Health Assessment Tool (AI)

## Problem Statement
Small and medium enterprises (SMEs) struggle to understand their financial health due to complex financial data.
This project provides a simple AI-based financial health assessment tool to analyze income, expenses, savings,
and loan obligations and give actionable insights.

## Features
- Analyze income vs expenses
- Identify savings health
- Detect high loan burden
- Simple AI-based rule engine
- Easy to understand financial recommendations

## Tech Stack

## How It Works
1. User provides financial data (income, expenses, loans, savings)
2. The system evaluates financial health using predefined rules
3. Outputs financial status and recommendations
