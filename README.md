# Financial Health Assessment Tool (AI)

This project analyzes income, expenses, savings, and loans to assess financial health.

## Features
- Income vs expense analysis
- Savings evaluation
- Loan burden detection
- Simple rule-based AI logic

## Technology
- Python

## How to Run
python financial_health.py

## Example
Input:
Income = 50000
Expenses = 30000
Savings = 15000
Loans = 10000

Output:
Good financial health
